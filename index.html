Welcome to the website of Mike DiLorenzo, husband, father, hiker, mountain biker, cross country skier, mountain enthusiast, craft beer drinker, web developer specializing in the craft of front end web development, and overall nice guy. This web site chronicles some of the more interesting things I've done such as my through hike of the Continental Divide trail, my four hikes of the Ring the Peak trail, my six hikes of Eldorado Mountain, and some back breaking landscaping projects that I really enjoy. The site will be updated when I do cool things, so don't be discouraged if it isn't updated for a few months, life can be uncool some times. But don't dispair! Coolness will prevail.
